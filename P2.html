        DATATYPE
        1)PRIMITIVE DATA TYPES
        2)NON-PRIMITIVE/DERIVED DATA TYPES

        1)PRIMITIVE DATA TYPES
            -INT(INTEGER)->it accept only round numbers EG-0-9.
            -FLOAT->it accept decimal value EG-10.5
            -BOOLEAN->either true or false
            -CHAR->it accept single character eg:-A,a.
            -VOID->returns null value
            

            2)NON PRIMITIVE/DERIVED DATA TYPES
            - ARRAY
            - SRTUCTURE
            - UNION
            - ENUM
            - POINTER
            - FUNCTION
